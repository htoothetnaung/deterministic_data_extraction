# Optional Mistral OCR parser adapter:
# No extra Python package is required; the backend calls Mistral's documented
# REST OCR endpoint directly. Set either env var:
# MISTRAL_API_KEY=<your key>
# EXTRACT_MISTRAL_API_KEY=<your key>

# Optional PaddleOCR parser adapters:
# Local OCR: install a PaddlePaddle build appropriate for your CPU/GPU first,
# then install paddleocr. For CPU quick tests this is usually:
#   pip install paddlepaddle paddleocr
# For GPU, install the CUDA-matched paddlepaddle-gpu build first, then set:
#   EXTRACT_PADDLEOCR_USE_GPU=true
# Optional explicit device override:
#   EXTRACT_PADDLEOCR_DEVICE=gpu:0
# PaddleOCR-VL API: install langchain-paddleocr and configure:
#   EXTRACT_PADDLEOCR_VL_API_URL=<endpoint>
#   AISTUDIO_ACCESS_TOKEN=<token>
# The adapter also accepts EXTRACT_AISTUDIO_ACCESS_TOKEN, PADDLEOCR_ACCESS_TOKEN,
# EXTRACT_PADDLEOCR_ACCESS_TOKEN, or the AI Studio token cache file.
# See docs/paddle_ocr/paddleocr_vl_setup.md for the brillar env compatibility
# note before changing the shared LangChain dependency line.
